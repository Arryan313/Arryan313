### 🤖 Intelligent Task Manager — ML-Powered Full-Stack App

> *Flask × scikit-learn × SQLite × Bootstrap*

An end-to-end intelligent task management system that uses Machine Learning to auto-prioritize tasks based on description text and deadline urgency.

**What I Engineered:**

| Component | Description |
|-----------|-------------|
| 🧠 ML Engine | Naive Bayes text classifier + deadline urgency scoring |
| 🔐 Auth System | Secure registration/login with Werkzeug password hashing |
| 🗄️ Database | Flask-SQLAlchemy ORM with relational task-user mapping |
| ⚡ Live Preview | AJAX-powered priority prediction before task submission |
| 📊 Dashboard | Real-time stats, smart filtering, and overdue detection |

**ML Technical Highlights:**
- **Weighted Scoring:** 60% text prediction + 40% deadline urgency = final priority
- **Keyword Detection:** `critical`, `asap`, `urgent` → High | `review`, `update` → Medium | `later`, `someday` → Low
- **Deadline Boost:** < 24h = +2 priority | < 72h = +1 | > 7 days = −1
- **Retraining Pipeline:** Model supports future retraining on real user data
